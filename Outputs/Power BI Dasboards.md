This folder contains the final dashboard deliverables for the NYC Taxi Operations & Revenue Intelligence project.

Due to GitHub storage limits and to reflect a production-style workflow, the Power BI file and dashboards are hosted externally in AWS S3 rather than stored directly in this repository.

### 📈 Power BI Dashboards
- [Power BI Report File (.pbix)](https://awale-nyc-taxi-project.s3.us-east-2.amazonaws.com/Data.pbix))

Interactive dashboard built in Power BI featuring:
- KPI overview (Trips, Revenue, Efficiency)
- Time-based demand and revenue analysis
- Borough-level performance comparisons
- Payment behavior insights
- Data quality diagnostics

> Open the `.pbix` file in Power BI Desktop to explore the full interactive experience.

---

### 🧠 Notes

- The dashboard is designed for **executive consumption**, prioritizing clarity, speed, and actionable insights.
- The report translates dashboard findings into **business decisions and strategic recommendations**.
- Both outputs were developed as part of a rapid, end-to-end analytics workflow using cloud-hosted data.
